Athena AI – Virtual Personal Assistant

Athena AI is an intelligent virtual assistant developed using Python and Streamlit. The application helps users manage daily tasks, take notes, access weather information, and interact through an AI-powered chat interface.

💬 AI Chat Assistant
✅ Task Management
📝 Notes Manager
🌦️ Real-Time Weather Updates
🎯 User-Friendly Dashboard
🔒 Secure and Lightweight Design
